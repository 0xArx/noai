# No AI Style Guide

All no AI programs are the same, regardless of use case. Any prompt you write is a liability.

## File Extensions

No AI code is not generated by models, but if you must label it, use the `.noai` file extension.

```
main.noai
```

## Linters

There is only one way to write no AI code and the `du` command can help you identify any issues.

```
du -h main.noai
```

```
0       main.noai
```

> The only valid output is human output.

## Code Reviews

The no AI community has adopted the following conventions when reviewing code changes:

When the change contains no AI-generated additions or modifications:

```
LGTM # Looks Good To Me
```

When the change includes AI-generated additions or modifications:

```
AIAL # AI Is A Liability
```

> Change requests that fall into this category should be rejected immediately.
